Conditions

Control structures are build with if - else if - else keywords followed by conditions that should be met true in order for the code to execute.
When you see the triple equals or three equal signs ===, then what that means is absolutely equals.

let attendance = 14;
if (attendance < 15) {
console.log("We have room");
} else if (attendance > 15) {
console.log("We're over capacity");
} else {
console.log("We're at full capacity");
}

Complex conditions

There's a way to have multiple conditions - two special operators that we can use to combine conditions for a single if or else if statement.

We have "and" represented as two ampersands && and we have "or" represented as two pipe characters ||.

A truth table for AND:

  • true and true equals true
  • true and false equals false
  • false and true equals false
  • false and false equals false
A truth table for OR:
  • true or true equals true
  • true or false equals true
  • false or true equals true
  • false or false equals false
const car = {
price: 2000,
color: "red",
numDoors: 4
};

if (car.price < 2000 || (car.color === "red" && car.numDoors === 4)) {
console.log("We'll take it!");
} else {
console.log("Welp, it's best to keep looking");
}
