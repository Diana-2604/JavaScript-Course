Objects in JavaScript

Objects are a special data type in JavaScript. They allow us to describe things with a lot of detail - the details are what we will call properties.
The properties allow us to describe our objects.

Creating Objects - Object Literal

Let's make a new variable called cookie.

We're going to assign it the value of empty curly braces {}. With these two empty curly braces, we have an object literal.

const cookie = {
name: "Chocolate Chip",
isGlutenFree: false,
};

Note: when working with objects in JavaScript - the order of output is not guaranteed.

The . (dot) operator is used to access properties of an object.

The object properties are kind of like variables in a way that you can actually assign them values using the equal sign:

cookie.name = "Chip Chocolate";

You can also access properties on objects by using that with the square brackets [].
The square brackets allow you to access properties via dynamic names or names with special characters:

console.log(cookie.["+Yummy"]);

Object literals are great for representing unique structures but when we need to repeat that structure Classes are better.

How to create objects using classes

Classes are a type of template to create our objects.

We're going to use the keyword class, then type Cookie with a capital C because that is the standard for making classes in JavaScript

We create a variable called myCookie, then we assign it the value of a new Cookie.

The new keyword creates a new object based on the class's structure.

class Cookie {}

const myCookie = new Cookie();

How to create a constructor

The constructor is just the rules that you need to create your class.

Constructors allow you to set the values for the properties of an object during creation.

So we want to say this.name, and when we say this in our constructor -
well, we're talking about this new object that we're creating. That's the context we need to keep.

class Cookie {
constructor(name, isGlutenFree) {
this.name = name;
this.isGlutenFree = isGlutenFree;
}
}

const myCookie = new Cookie("Iced Oatmeal", false);

How to add functions to object literals and classes

In our object literal, we're going to add a function.

The property name should be eatCookie - this should be a verb as opposed to a noun. It should be an action.

We're going to assign it the value of a function.

const cookie = {
name: "Chocolate Chip",
isGlutenFree: false,
eatCookie: function () {
console.log("I am eating " + this.name);
}
};
cookie.eatCookie;


Let's do the same thing in our Cookie class.

To accomplish this, we will create what they call a method, which has a function that belongs to a class.

Inside of the class, you don't need the function keyword.

class Cookie {
constructor(name, isGlutenFree) {
this.name = name;
this.isGlutenFree = isGlutenFree;
}

eatCookie() {
console.log("I am eating " + this.name);
}
}

const myCookie = new Cookie("Iced Oatmeal", false);
myCookie.eatCookie();
