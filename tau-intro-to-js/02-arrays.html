Arrays in JavaScript

Array is a data structure to store multiple values in the same variable
Type const array = []; to create an empty array
Array indices start at 0

Adding values to an array

There are two ways that you can add values to an array: assignment and push

Assignment
When you create the array, you can add its elements inside the square brackets [] after the equal sign

const favoriteCookies = [
"chocolate chip",
"iced oatmeal",
"sugar cookie",
"butter cookie"
];

Push
You type the name of your variable - it must be an array.
Then type .push(), and whatever goes inside of these parentheses will be added as the last item of your array.
This is really dynamic because what you can do is you can get data maybe from a third party data source and add elements to an array.

Removing items from an array

To remove items, what you do is use the pop command.
The .pop() is going to remove the last element in the array.
You can actually store the value that was popped off the array:

let cookie = favoriteCookies.pop();

Accessing an individual item in an array

We have to use the square bracket notation [] with the element index inside:

console.log(favoriteCookies[0]);

In case you go out of bounds of what the indice range is, in JavaScript you'll get undefined

I can also put a variable fav here and set it to an index value. It's going to replace the fav with the actual value.

let fav = 3;
console.log(favoriteCookies[fav]);

How many elements are in an array?

Arrays have a property called .length.

console.log(favoriteCookies.length);

Since the array goes from 0 to the length-1, to access the last element type:

console.log(favoriteCookies[favoriteCookies.length-1]);
