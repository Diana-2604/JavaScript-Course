Variables in JavaScript

In JavaScript, you have three ways to declare a variable:

  1. the let keyword
  2. use the const keyword
  3. you also have access to the var keyword
Now, we don't really use var as much before because that's more of a legacy feature for JavaScript.
var has some very different rules than let and const.

let allows you to create a variable, but reassign the value,

whereas const - when you create a variable, you actually can't reassign that value.

JavaScript Data Types

JavaScript has the concept of primitive data types: strings, numbers, boolean and undefined/null.

Strings are letters, numbers, emojis, and any type of character you see on the keyboard.

Numbers can be positive, negative whole numbers, or even be numbers with decimals.
Numbers are great because you can do basic operations on them like add, subtract, multiply, divide.

Boolean - the boolean type is strictly true or false.

Undefined usually means that there is no value.
If I created a new variable without assigning an initial value, I'll have undefined.

Null usually means that we are clearing out the value.
If a variable is set to null that usually means that we're saying it doesn't have a value.
